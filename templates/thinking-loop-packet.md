# Thinking Loop Packet

Use this packet when the task is synthesis, framing, judgment formation, critique, or strategy.

## Tension

State the unresolved tension, question, or decision pressure.

## Current Context

Record the current project, conversation, source, or external signal that triggered the thinking work.

## Observations

List what is directly observed from current sources.

## Inferences

List what is inferred from the observations. Keep inference separate from fact.

## Judgment

State the current judgment and its scope.

## Counterargument

Write the strongest competing interpretation.

## Decision Candidate

State the decision this thinking suggests, if any.

## Parking Lot

Move useful but non-current ideas to:

- P1;
- P2;
- decision later;
- phase research.

## Output

Choose where the synthesis should land:

- direct answer only;
- local process decision entry;
- project document;
- research backlog;
- memory candidate;
- external expression later.
